# Мережевий TCP-застосунок

Цей проєкт є реалізацією клієнт-серверної архітектури на основі протоколу TCP для операційних систем сімейства Linux та macOS. Програма дозволяє кільком клієнтам одночасно взаємодіяти з сервером, обмінюючись текстовими повідомленнями.

## Функціональні можливості

* **TCP Сервер:**
    * Підтримує одночасне підключення кількох клієнтів за допомогою багатопотоковості (`std::thread`).
    * Реагує на повідомлення "hello" відповіддю "world".
    * Повертає будь-які інші повідомлення клієнту без змін.
    * Коректно завершує сесію при отриманні команди "disconnect".
* **TCP Клієнт:**
    * Встановлює стабільне з'єднання з сервером за вказаною IP-адресою та портом (за замовчуванням `127.0.0.1:8080`).
    * Дозволяє користувачеві вводити повідомлення в інтерактивному режимі.
    * Завершує роботу при введенні команди "disconnect".

## Архітектура та керування ресурсами

Проєкт розроблений з акцентом на надійність та безпеку роботи з пам'яттю:

1.  **Багатопотоковість:** Для кожного нового підключення сервер створює окремий потік виконання, що дозволяє не блокувати головний цикл очікування нових клієнтів.
2.  **RAII (SocketGuard):** Реалізовано спеціальну структуру `SocketGuard`, яка гарантує автоматичне закриття файлових дескрипторів сокетів при виході з області видимості. Це запобігає витоку системних ресурсів.

## Системні вимоги

* Операційна система: Linux або macOS.
* Компілятор: `g++` або `clang++` з підтримкою стандарту C++17.
* Утиліта збірки: `make`.

## Інструкція зі збірки та запуску

Для автоматизації процесу компіляції використовується `Makefile`.

### 1. Компіляція проєкту
Відкрийте термінал у папці з проєктом та виконайте команду:
`make`
Це створить два виконувані файли: `server` та `client`.

### 2. Запуск сервера
Запустіть серверний додаток у першому вікні термінала:
`./server`
За замовчуванням сервер слухає порт `8080` та готовий приймати підключення.

### 3. Запуск клієнта
У новому вікні термінала (можна відкрити 2-3 вікна одночасно для тестування багатопотоковості) запустіть клієнт:
`./client`

## Використання (Тестування)
1. Після успішного підключення введіть текст у консолі клієнта.
2. Напишіть `hello` — сервер розпізнає команду і у відповідь надішле `world`.
3. Напишіть будь-який інший текст (наприклад, `test message`) — сервер поверне його назад без змін (Echo).
4. Введіть `disconnect` — клієнт повідомить сервер, розірве TCP-з'єднання та завершить свою роботу. Сервер при цьому коректно звільнить пам'ять для цього клієнта і продовжить чекати на нові підключення.
5. Щоб повністю зупинити сервер, натисніть `Ctrl + C` у вікні термінала, де він працює.

## Структура проєкту
* `server.cpp` — вихідний код TCP-сервера.
* `client.cpp` — вихідний код TCP-клієнта.
* `Makefile` — скрипт для автоматичної збірки та компіляції проєкту.
* `README.md` — технічна документація проєкту.
